# sales-analysis
A Python-based Sales Intelligence Dashboard that covers the complete data analytics pipeline — data generation, visualization, machine learning, and Power BI integration.
It analyzes 2000 sales transactions across regions, categories, and segments using Matplotlib, Seaborn, and Plotly. Includes a Revenue Forecasting model and Customer Segmentation using Scikit-learn, with a Power BI ready Excel export.
Skills: Python · Data Visualization · Machine Learning · Power BI
